2. Erhebung und Verarbeitung von Daten

Wir erheben und verarbeiten Daten in folgenden Fällen:

  • Hosting (Cloudflare Pages): Unsere Website wird über Cloudflare Pages bereitgestellt. Cloudflare verarbeitet dabei technisch notwendige Log-Daten (IP-Adresse, Datum, Uhrzeit, aufgerufene Seite, Browser-Typ). Diese Daten dienen der Sicherheit und dem störungsfreien Betrieb. Datenschutzerklärung Cloudflare: cloudflare.com/privacypolicy
  • Audio-Hosting (Infomaniak, Schweiz): Unsere Audiodateien sind auf S3-kompatiblem Objektspeicher der Infomaniak Network SA in der Schweiz gespeichert. Beim Abspielen einer Datei wird Ihre IP-Adresse technisch bedingt übertragen. Infomaniak ist ein Schweizer Anbieter; die Daten verbleiben in der Schweiz. Datenschutzerklärung Infomaniak: infomaniak.com
  • Kontaktformular (Formspree): Wenn Sie das Kontaktformular verwenden, werden Ihre Angaben (Name, E-Mail-Adresse, Nachricht) an den Dienst Formspree übermittelt und von dort an unsere E-Mail-Adresse weitergeleitet. Die Daten werden ausschliesslich zur Bearbeitung Ihrer Anfrage verwendet und danach gelöscht. Datenschutzerklärung Formspree: formspree.io/legal/privacy-policy
  • Google Fonts: Diese Website lädt Schriftarten von Google Fonts. Dabei wird Ihre IP-Adresse an Google-Server übertragen. Datenschutzerklärung Google: policies.google.com/privacy
  • Webanalyse (Cloudflare Web Analytics): Diese Website nutzt Cloudflare Web Analytics zur Analyse des Datenverkehrs. Cloudflare Web Analytics verwendet keine Cookies und erhebt keine personenbezogenen Daten. Es werden ausschliesslich aggregierte, anonymisierte Metriken (z.B. Seitenaufrufe, Herkunftsland) erfasst. Ein Cookie-Banner ist nicht erforderlich. Datenschutzerklärung Cloudflare: cloudflare.com/privacypolicy
  • Zahlungsabwicklung (Payhip): Käufe über den Shop werden über Payhip Ltd. abgewickelt. Beim Klick auf einen Kaufen-Button werden Sie auf eine von Payhip gehostete Checkout-Seite weitergeleitet. Dort gelten die Datenschutzbedingungen von Payhip. Wir erhalten von Payhip keine vollständigen Zahlungsdaten (z.B. keine Kreditkartennummern). Datenschutzerklärung Payhip: payhip.com/privacy

3. Cookies

Diese Website verwendet keine Tracking-Cookies. Es können technisch notwendige Session-Cookies des Hosters (Cloudflare) gesetzt werden, die für den Betrieb der Website erforderlich sind.

4. Datenspeicherung in der Schweiz

Wir legen grossen Wert auf die Datenspeicherung in der Schweiz. Unsere Audiodateien liegen auf Servern der Infomaniak Network SA, einem Schweizer Unternehmen mit Sitz in Genf. Die Website wird über Cloudflare Pages bereitgestellt, wobei Cloudflare weltweit verteilte Rechenzentren nutzt.
